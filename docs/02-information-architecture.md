## User Journeys

### Journey 1: New Visitor via Meta Ads (Primary Revenue Driver)

```
Meta Ad (Lifestyle Video / Image)
    → Landing Page (Hero + Wallet Collection)
    → Product Page (Essential Wallet $49)
    → Add to Cart
    → Cart Drawer (Upsell: Leather Care Kit $19)
    → Checkout
    → Order Confirmation
    → Post-Purchase Email (Bag Collection Teaser)
    → Return Visit (30-60 days)
    → Premium Bag Purchase ($280–$450)
```

### Journey 2: Organic Discovery (Brand Searcher)

```
Google Search "luxury leather bags for travel"
    → Homepage
    → Navigation → Collections → Travel Bags
    → Product Page (The Weekender $395)
    → Browse Related Products
    → Add to Wishlist (if not ready)
    → Email Capture (Newsletter popup)
    → Return via Email
    → Purchase
```

### Journey 3: Returning Customer

```
Email (New Arrival / Journal Article)
    → Journal Article (Travel Inspiration)
    → In-Article Product Links
    → Product Page
    → Quick Add to Cart (Already has account)
    → Express Checkout
```

### Journey 4: Gift Buyer

```
Google "luxury leather wallet gift for men"
    → Collection Page (Wallets & Cardholders)
    → Product Page
    → Gift Wrapping Option at Cart
    → Add Personal Note
    → Checkout
```

## Page Blueprints

### Homepage Blueprint

| Section | Purpose | Conversion Goal |
|---------|---------|-----------------|
| 1. Hero | Emotional hook, brand positioning | Click to collection |
| 2. Brand Values | Trust + differentiation | Build credibility |
| 3. Featured Collection | Product discovery | Product page visits |
| 4. Brand Story | Emotional connection | Brand buy-in |
| 5. Lifestyle Gallery | Aspiration + social proof | Engagement |
| 6. Craftsmanship | Quality perception | Justify pricing |
| 7. Testimonials | Social proof | Trust building |
| 8. Newsletter | Email capture | List growth |
| 9. Footer | Navigation + trust | Site exploration |

### Product Page Blueprint

| Section | Purpose |
|---------|---------|
| Gallery (5-7 images) | Visual selling, zoom capability |
| Product Title + Price | Clear identification |
| Variant Selector | Color/size choice |
| Add to Cart (Sticky) | Conversion action |
| Trust Strip | Shipping, returns, warranty |
| Accordion: Materials | Justify quality |
| Accordion: Dimensions | Practical info |
| Accordion: Craftsmanship | Brand story |
| Accordion: Shipping | Remove objections |
| Accordion: Returns | Remove risk |
| Lifestyle Section | Aspirational context |
| Related Products | Cross-sell |
| Recently Viewed | Re-engagement |
| Reviews | Social proof |

### Collection Page Blueprint

| Element | Purpose |
|---------|---------|
| Collection Hero | Category storytelling |
| Filter Bar | Navigation efficiency |
| Product Grid (3-col desktop, 2-col mobile) | Product browsing |
| Editorial Banners (mid-grid) | Break monotony, storytelling |
| Load More / Pagination | Performance |

## Mobile Experience Priorities

1. Thumb-zone navigation
2. Full-width product images
3. Sticky Add to Cart on product pages
4. Simplified navigation (hamburger → full-screen overlay)
5. Touch-optimized filter drawers
6. Swipeable product galleries
7. Bottom-anchored CTA buttons
8. Reduced animation for performance
9. Optimized image loading (WebP, lazy load)
10. Single-column layouts everywhere except product grid (2-col)
